NAME:

	boks.py: a book/author/publication database navigator

SYNOPSIS:

	usage: python3 books.py [-h] [-u] [ts string] [-as string] [-yrs year1 year2]

Optinal arguments:

-h, --help
	Shows this help message
-ts string [year] [publication], --titlesearch string [year] [title]
	Searches for and prints a list of books whose titles contain the string argument, case insensitive. May be
	sorted by publication year or by title
-as string, --authorsearch string
	Searches for and prints a list of books whose authors contain the string argument, case insensitive. Sorted
	alphabetically by surname
-yrs year1 year2, --yearsearch year1 year2
	Searches for and prints a list of books whose publication year lies between the year1 and year2 arguments.
	Sorted by publication year ascending.
-u, --usage
	Prints usage statement above.
