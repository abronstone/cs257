VIDINFO





VIDINFO is a web application designed by Jack Owens and Aaron Bronstone in CS257: Software Design at Carleton College. This application supports the following four pages:

1.) SEARCH: search for movies based on custom filters!

2.) OVERVIEWS: get a detailed overview of a particular movie!

3.) POPULARITY: get a list of the most popular movies, directors, actors, genres, and much more!

4.) COMPARISON: get a comparison of common competitive values between movies, such as rating, popularity, budget and revenue!
