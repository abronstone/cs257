VIDINFO





VIDINFO is a full stack web application containing a database of over 45,000 movies released prior to 2017!

This application supports the following four pages:

1.) SEARCH: search for movies based on custom filters!

2.) OVERVIEWS: get a detailed overview of a particular movie!

3.) MOVIE GENERATOR: get o detailed overview of a randomly selected movie!

4.) COMPARISON: get a comparison of common competitive values between movies, such as rating, popularity, budget and revenue!

Note: This application uses a PSQL database derived from 'The Movies Dataset' on Kaggle:
https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset
